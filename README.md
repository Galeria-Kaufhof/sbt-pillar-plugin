# sbt-pillar-plugin - manage migrations for your Cassandra data stores via sbt

[![Build Status](https://travis-ci.org/inoio/sbt-pillar-plugin.png?branch=master)](https://travis-ci.org/inoio/sbt-pillar-plugin)
